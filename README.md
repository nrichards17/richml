richml
======

Currently, this package includes:
- custom PyTorch schedulers
    * Cosine annealing w/ restarts
    * 1-Cycle
    * 1-Cycle w/ restarts

Dependencies
------------

Found in `setup.py`
```
INSTALL_REQUIRES = [
    'numpy>=1.15.0',
    'torch>=0.4.1',
]
```

Some examples also require  `matplotlib`

Installation
------------

    pip install git+https://github.com/nrichards17/richml#egg=richml
